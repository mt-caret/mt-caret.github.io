Module Polars.Env

There are many environment variables that effect Polars behavior, mostly related to the display of tables. This module is responsible for setting the default values of these variables so that e.g. expect tests will be less fragile.

See https://docs.rs/polars/latest/polars/#config-with-env-vars and https://github.com/pola-rs/polars/blob/9c194a24ad6aac33004c8fb0515a90239a376f44/py-polars/polars/config.py for more details on which environment variables exist and what they mean.
